Rivero's Motor Shop Site Plan

Author: Emiliano Rivero

Course: WDD 231 – Individual Project

Site Name

Rivero's Motor Shop

Rivero's Motor Shop is an online showroom focused on high-performance sports cars and motorcycles. The name uses my last name as the brand to make the shop feel personal, trustworthy, and memorable.

Optional domain: riverosmotorshop.com

Site Purpose

The purpose of Rivero's Motor Shop is to showcase and sell high-performance sports cars and motorcycles. The website will:

  • Display a catalog of cars and bikes with clear specs, images, and prices.
  • Allow visitors to filter vehicles by type, brand, and budget.
  • Provide contact and request forms for test drives and questions.
  • Explain basic financing and payment options offered by the shop.

Scenarios

  • Scenario 1: “I want to see which sports bikes are available under $15,000.”
  • Scenario 2: “How can I contact Rivero's Motor Shop to schedule a test drive?”
  • Scenario 3: “Does the shop offer financing or payment plans for cars and motorcycles?”

Color Scheme

The site uses a dark, sporty color palette to match the feeling of high-performance vehicles. The main background is a deep, almost-black blue, with a bright red accent that highlights important actions like buttons and links. Light gray is used for text and cards.

Color Hex Value Usage
Primary Background #0b0c10 Page background, header background.
Accent #e63946 Buttons, links, hover effects, key highlights.
Neutral Light #f5f5f5 Main text color on dark backgrounds, card backgrounds.
Secondary Dark #1f2833 Section backgrounds, cards, navigation bar.

Typography

The site uses two main fonts from Google Fonts. Oswald is used for headings to give a bold, sporty look, and Roboto is used for body text to keep the content easy to read.

Font Usage Example
Oswald Headings (h1, h2, h3), logo text. RIVEROS MOTOR SHOP
Roboto Body text, paragraphs, lists, buttons. High-performance cars and motorcycles for true enthusiasts.

Wireframe Sketches

These wireframes show the planned layout for the Rivero's Motor Shop home page on mobile and on larger (desktop/tablet) viewports.

Mobile Layout Description

  • Header with logo and hamburger menu.
  • Hero image with a main call-to-action button.
  • Featured cars section with vertical cards.
  • Featured motorcycles section.
  • Short “Why choose us?” section.
  • Footer with contact and basic information.

Desktop Layout Description

  • Header with logo on the left and navigation links on the right.
  • Hero section with text and button on the left, large vehicle image on the right.
  • Grid of featured cars (3 columns).
  • Grid of featured motorcycles (3 columns).
  • Three-column “Why Rivero's Motor Shop?” section.
  • Footer with contact info and links.

Testing and Evaluation

This site plan and the future project pages will be tested with:

  • W3C HTML Validator for HTML markup.
  • WAVE for accessibility checks.
  • Chrome DevTools Lighthouse for performance, best practices, and SEO.
  • WebAIM Contrast Checker to verify color contrast.
